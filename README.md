## Installation

1. Install

   ```sh
   npm install
   ```

2. Start the development server

   ```sh
   npm run dev
   ```

## Build and Run for Production

1. Generate production build

   ```sh
   npm run build
   ```

1. Preview the site once deployed

   ```sh
   npm run start
   ```
